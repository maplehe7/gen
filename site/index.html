GitHub Pages Build Console

Dispatch Unity exports from a static site.

This page runs on GitHub Pages. It does not execute Python in the browser. Instead, it triggers a GitHub Actions workflow that runs unity_standalone.py, publishes the generated Build/ and index.html, and returns a play link when the workflow finishes.

New Export

Enter a direct game URL or a name from game_catalog.json. The repo target and token are hardcoded for this demo in site/config.js.

How It Works

  1. The page resolves the input to a source URL.
  2. It dispatches .github/workflows/build-game.yml.
  3. The workflow runs the Python exporter in GitHub Actions.
  4. The completed folder is deployed back to GitHub Pages.

Required token scope in site/config.js: Actions: read and write and Contents: read and write.
