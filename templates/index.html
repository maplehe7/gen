Unity Standalone Wrapper

Turn a game URL into a hosted standalone build.

Paste a supported game URL or a name from your local catalog. The app runs unity_standalone.py, tracks progress, and serves the generated Build/ folder plus launcher HTML when the export finishes.

New Export

Name mode resolves through game_catalog.json. Direct links work immediately.

Pipeline

  1. Resolve the submitted URL or catalog match.
  2. Run the standalone exporter in a background job.
  3. Track progress from logs and the exporter progress file.
  4. Expose the final launcher at a permanent play URL.
